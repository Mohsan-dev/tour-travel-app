# Trippy - Tour & Travel App

A responsive tour and travel website built with React.js.

## Features
- Responsive Navbar with hamburger menu
- Dynamic Hero section
- Popular Destinations section
- Trip packages section
- Footer
- Contact form
- About page
- Sign Up page

## Tech Stack
- React.js
- React Router DOM
- CSS
- Font Awesome
