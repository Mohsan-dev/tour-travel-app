# 🌍 Trippy - Tour & Travel App

A modern, responsive tour and travel website built with React.js. Plan your next adventure with style!

## 🚀 Live Demo
https://tour-travel-webapp.netlify.app/

## ✨ Features

* 🧭 Responsive Navbar with hamburger menu
* 🏔️ Dynamic Hero section with stunning visuals
* 📍 Popular Destinations showcase
* 🎒 Trip packages section
* 📝 Contact form
* ℹ️ About page
* 🔐 Sign Up page
* 📱 Fully responsive across all devices

## 🛠️ Tech Stack

* React.js
* React Router DOM v6
* CSS3 (Responsive Design)
* Font Awesome Icons
* Vite (Build Tool)

## ⚙️ How to Run Locally

1. Clone repository:
```bash
git clone https://github.com/Mohsan-dev/tour-travel-app.git
```

2. Open folder:
```bash
cd tour-travel-app
```

3. Install dependencies:
```bash
npm install
```

4. Start dev server:
```bash
npm run dev
```
